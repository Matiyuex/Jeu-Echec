# Jeu-Echec
Jeu d'échec réalisé lors d'un projet scolaire.
Ce jeu comprend toutes les règles des éches classique (promotion, roque, prise en passant, echec, echec et mat).
Celui-ci se joue dans l'invité de commandes en rentrant la pièce à déplacer (ex : e2) puis en rentrant la destination selon les règles de la pièce (ex : e4).
